## 3. On-Site: Diagnostic Process

### Standard Diagnostic ($111)
1. Collect $99 dispatch on arrival (or confirm collected)
2. Perform full system inspection — do NOT skip steps to save time
3. Diagnose in **chronological order** — explain what failed first, what caused what
4. Document findings in HCP with photos
5. Build repair quote in HCP before leaving
6. Present quote to customer — explain each line item
7. Collect $111 diagnostic fee
8. Inform customer: **diagnostic credited toward repair if approved within 14 days**
9. If customer approves on-site: begin repair, apply $111 credit to invoice
10. If customer needs time: leave written quote, follow up in 3 days

### What to Photo Every Visit
- Equipment nameplate (model/serial)
- Problem area (burned contacts, leak, cracked component)
- Before and after if repair done
- System overview shot

## 4. Repair

1. All parts pulled from truck stock or ordered same-day if available locally
2. Complete repair, test system through full cycle
3. Document in HCP: parts used, labor time, findings
4. Invoice in HCP — apply $111 diagnostic credit if within 14-day window
5. Collect payment before leaving (card via HCP, check, or cash)
6. Ask for Google review **verbally** before leaving: *"If we did good work today, I'd really appreciate a Google review — it helps a lot."*
7. HCP auto-sends review request text after job marked complete

## 6. New System Install / Estimate

### Estimate Process
1. Schedule free estimate (no dispatch, no diagnostic fee)
2. Perform Manual J load calculation for new installs
3. Build quote in HCP — present 3 options (good/better/best) when possible
4. Include: equipment, labor, permits if needed, warranty
5. For geo: include utility rebate amount for their specific utility
6. Leave written quote — follow up in 3 days if no decision
7. Offer financing (Synchrony, Wells Fargo, Wisetack) on every estimate

### Install Day
1. Confirm arrival time morning of
2. Protect floors/work area
3. Pull permit if required (Kingfisher and surrounding cities)
4. Photo documentation throughout
5. Walk customer through new system operation before leaving
6. Register equipment warranty same day
7. Enter in HCP with system info, photos, warranty date
8. Ask for Google review on-site
9. Offer PMA enrollment at time of install
